# Sprangular

Spree + Angular.js frontend

## Installing

Add `sprangular` and rails-assets source to your Gemfile

```
source 'https://rails-assets.org'
gem 'sprangular'
```

Mount the engine in your `config/routes.rb`

`mount Sprangular::Engine => "/"`
